## 1. Google Business Profile セットアップ

### 🎯 なぜやるか
Google マップ / Google 検索（「○○ 花屋」）で出てくる**店舗情報カード**。
**地域名検索の流入が3〜10倍**になる超重要な施策。無料。

### ⏱ 所要時間: 1時間（最初の登録 30分 + オーナー確認 待ち時間あり）

---

### Step 1: アカウント作成・登録（10分）

1. https://www.google.com/intl/ja_jp/business/ にアクセス
2. 「今すぐ管理」をクリック → Google アカウントでログイン
3. **店舗名を入力**（FLORIX管理画面の店舗名と完全一致させる）
4. **業種**: 「生花店」または「フローリスト」を選択
5. **住所を入力**: 郵便番号 → 都道府県 → 市区町村 → 番地
6. **配達範囲を設定**: 配達対応エリアを地図上でドラッグして指定
7. **電話番号**: 店舗の電話番号
8. **ウェブサイト**: `https://shop.七社さまのドメイン.com`（FLORIXのURL）
   - もしくは `https://www.noodleflorix.com/order/[tenantId]/[shopId]`

### Step 2: オーナー確認（待ち時間: 2〜14日）

Google から **ハガキ** が届きます（通常 1〜2週間）。
ハガキに書かれた **5桁の確認コード** を Google Business Profile 管理画面で入力。
→ これで認証完了。

電話確認や動画確認が選べる場合もあるので、選べる時はそちらが早い。

### Step 3: 必須情報をすべて埋める（30分・最重要）

| 項目 | やること |
|------|---------|
| **営業時間** | 月〜日の正確な時間。定休日も明示。 |
| **特別営業時間** | お盆・年末年始・GW を必ず設定 |
| **店舗の説明** | 750文字以内・最初の250文字に重要キーワード |
| **店舗写真** | 外観1・店内3・スタッフ1・商品5〜10枚 |
| **ロゴ** | FLORIX管理画面で使ってるロゴと同じものを |
| **カバー写真** | 横長で映える1枚（例: 看板商品の花束） |
| **属性** | 「配達あり」「Wi-Fiあり」等チェック |
| **サービス** | 「花束」「アレンジメント」「スタンド花」等を個別に追加 |
| **商品** | EC商品の上位3〜5件をGBPにも掲載 |

### Step 4: 投稿機能の活用（週1運用）

GBP内の **「投稿」機能** で:
- 新商品入荷
- 季節のキャンペーン（母の日・お盆・敬老の日・クリスマス）
- イベント情報（フラワーレッスン等）

→ Google 検索結果カードに表示されるので **超効果的**。

### Step 5: クチコミ管理（最重要・継続）

- **クチコミに必ず返信**（24時間以内が理想）
- ポジティブ: 感謝＋具体的にどんな商品か再言及（キーワード仕込み）
- ネガティブ: 真摯に謝罪・改善案提示（他のお客さんが見てる）

📌 **クチコミお願いカード**を商品に同封すると効果絶大
→ FLORIX管理画面で「クチコミURLをQRコード化」してカードに印刷

GBP管理画面 → 「クチコミ」→「短縮URL」をコピー → QR化

## 2. Google Search Console 改善ループ

### 🎯 なぜやるか
**今すでに来てるアクセス** を分析して、もう少し改善するだけで上位表示が狙えるキーワードを発見する。
**月1回・30分** で十分。

### 📊 週次チェック（毎週月曜・5分）

Google Search Console: https://search.google.com/search-console

1. **「検索パフォーマンス」** を開く
2. 期間を **「過去7日」** に設定
3. 4つの数字を確認:
   - **表示回数**: 検索結果に出た回数（増えてればOK）
   - **クリック数**: 実際にクリックされた回数
   - **CTR (%)**: クリック率 = クリック / 表示
   - **平均掲載順位**: Google検索での平均位置

### 📈 月次チェック（毎月1日・30分）

#### A. 「もう少しで上位」キーワードを発見

1. 期間 **「過去28日」**
2. **「クエリ」タブ**
3. 並び替え: **平均掲載順位 → 11〜30位** で絞り込み

→ **これが宝の山**。「もうちょい」で1ページ目（10位以内）に入る。

→ 該当キーワードを使っている商品ページの:
- **タイトル** を見直す
- **説明文** を厚く書く
- **画像 alt** を見直す

#### B. CTR が低いキーワードを改善

1. 表示回数100以上で CTR が **2%以下** のもの
2. → タイトルが魅力的じゃない or 検索意図とズレてる
3. → タイトル・descriptionを書き直す

#### C. インデックス状況チェック

1. **「ページ」** タブ
2. **「未登録」** をクリック
3. 重要なページが未登録だったら **「URLを検査」** → **「インデックス登録をリクエスト」**

#### D. Core Web Vitals

1. **「ウェブに関する主な指標」**
2. 「不良」「改善が必要」がないか確認
3. あれば該当URLを **しーちゃん** に連絡 → 修正対応

### 🔁 改善ループの流れ

```
①GSCで「11〜30位」を発見
   ↓
②該当キーワードで上位サイトを観察（何を書いてる？何が違う？）
   ↓
③自店ページのタイトル・説明・本文を改善
   ↓
④2週間〜1ヶ月後にGSCで順位確認
   ↓
⑤上がった → 次のキーワードへ / 上がらない → さらに改善
```

## 3. GA4 のチェックポイント

### 📊 月次チェック（5分）

Google Analytics: https://analytics.google.com

1. **「リアルタイム」**: 今アクセスしてる人がいるか
2. **「レポート → エンゲージメント → ページとスクリーン」**: 人気ページTOP10
3. **「レポート → 集客 → ユーザー獲得」**:
   - Organic Search（Google検索からの流入）の比率を確認
   - 60%以上が目標

### 🎯 コンバージョン（要設定）

- `purchase` (Stripe決済完了)
- `view_item` (商品ページ閲覧)
- `generate_lead` (見積依頼・カスタム注文送信)

→ GA4管理画面 → 「キーイベント」で設定（しーちゃんに依頼してOK）

## 💡 こだわりポイント・コツ

### キーワードの選び方

**× 大ざっぱ**: 「花」「花屋」（競争激しすぎ）
**◯ 具体的**: 「札幌 北区 花屋 即日配達」「胡蝶蘭 5本立 開店祝い 立札無料」

「地域名 + 業態 + 具体的なニーズ」の3要素を組み合わせる。

### 写真のSEO効果

- ファイル名を **日本語商品名** に変える（例: `pink-bouquet.jpg` → `ピンクのブーケ.jpg`）
- 画像の **alt属性** を必ず記述（FLORIX管理画面で設定）

### クチコミの伸ばし方

- お渡し時に「いかがでしたか？」と一言
- お礼カードにQRコード（GBPクチコミ画面に直行）
- 配達商品に「クチコミお願いカード」同封
